# Sample HTML & CSS Project

### **Project Description**

A simple static website built using only **HTML** and **CSS** to practice layout design, styling, and basic responsive web structure. The project showcases clean UI, simple sections, and a beginner-friendly front-end design approach.

### **Tech Stack Used**

* **HTML5**
* **CSS3**

### **Purpose**

Created as a practice project to improve front-end development skills and understand core HTML/CSS concepts.
